# `@peculiar/asn1-x509-microsoft`

[![License](https://img.shields.io/badge/license-MIT-green.svg?style=flat)](https://github.com/PeculiarVentures/asn1-schema/blob/master/packages/x509-microsoft/LICENSE)
[![npm version](https://badge.fury.io/js/%40peculiar%2Fasn1-x509-microsoft.svg)](https://badge.fury.io/js/%40peculiar%2Fasn1-x509-microsoft)

[![NPM](https://nodei.co/npm/@peculiar/asn1-x509-microsoft.png)](https://nodei.co/npm/@peculiar/asn1-x509-microsoft/)

ASN.1 schema for Microsoft-specific X.509 certificate extensions and object identifiers.

Use the exported classes with `@peculiar/asn1-schema` helpers such as `AsnConvert`, `AsnParser`, and `AsnSerializer` to parse or serialize DER-encoded data defined by the referenced specification.

## Installation

```bash
npm install @peculiar/asn1-x509-microsoft
```

## Specifications

- Microsoft-specific certificate template, policy, and extension identifiers.
