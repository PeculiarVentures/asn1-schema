# `@peculiar/asn1-pkcs8`

[![npm version](https://badge.fury.io/js/%40peculiar%2Fasn1-pkcs8.svg)](https://badge.fury.io/js/%40peculiar%2Fasn1-pkcs8)

[![NPM](https://nodei.co/npm/@peculiar/asn1-pkcs8.png)](https://nodei.co/npm/@peculiar/asn1-pkcs8/)

ASN.1 schema for PKCS #8 private key information structures defined in RFC 5208.

Use the exported classes with `@peculiar/asn1-schema` helpers such as `AsnConvert`, `AsnParser`, and `AsnSerializer` to parse or serialize DER-encoded data defined by the referenced specification.

## Installation

```bash
npm install @peculiar/asn1-pkcs8
```

## Specifications

- [RFC 5208: PKCS #8 Private-Key Information Syntax Specification](https://datatracker.ietf.org/doc/html/rfc5208)
