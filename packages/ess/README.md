# `@peculiar/asn1-ess`

[![License](https://img.shields.io/badge/license-MIT-green.svg?style=flat)](https://github.com/PeculiarVentures/asn1-schema/blob/master/packages/ess/LICENSE)
[![npm version](https://badge.fury.io/js/%40peculiar%2Fasn1-ess.svg)](https://badge.fury.io/js/%40peculiar%2Fasn1-ess)

[![NPM](https://nodei.co/npm/@peculiar/asn1-ess.png)](https://nodei.co/npm/@peculiar/asn1-ess/)

ASN.1 schema for Enhanced Security Services (ESS) for S/MIME defined in RFC 2634 and RFC 5035.

Use the exported classes with `@peculiar/asn1-schema` helpers such as `AsnConvert`, `AsnParser`, and `AsnSerializer` to parse or serialize DER-encoded data defined by the referenced specification.

## Installation

```bash
npm install @peculiar/asn1-ess
```

## Specifications

- [RFC 2634: Enhanced Security Services for S/MIME](https://datatracker.ietf.org/doc/html/rfc2634)
- [RFC 5035: Enhanced Security Services (ESS) Update](https://datatracker.ietf.org/doc/html/rfc5035)
