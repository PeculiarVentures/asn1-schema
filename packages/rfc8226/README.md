# `@peculiar/asn1-rfc8226`

[![npm version](https://badge.fury.io/js/%40peculiar%2Fasn1-rfc8226.svg)](https://badge.fury.io/js/%40peculiar%2Fasn1-rfc8226)

[![NPM](https://nodei.co/npm/@peculiar/asn1-rfc8226.png)](https://nodei.co/npm/@peculiar/asn1-rfc8226/)

ASN.1 schema for secure telephone identity credential certificates defined in RFC 8226.

Use the exported classes with `@peculiar/asn1-schema` helpers such as `AsnConvert`, `AsnParser`, and `AsnSerializer` to parse or serialize DER-encoded data defined by the referenced specification.

## Installation

```bash
npm install @peculiar/asn1-rfc8226
```

## Specifications

- [RFC 8226: Secure Telephone Identity Credentials: Certificates](https://datatracker.ietf.org/doc/html/rfc8226)
