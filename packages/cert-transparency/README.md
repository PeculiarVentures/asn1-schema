# `@peculiar/asn1-cert-transparency`

[![License](https://img.shields.io/badge/license-MIT-green.svg?style=flat)](https://github.com/PeculiarVentures/asn1-schema/blob/master/packages/cert-transparency/LICENSE)
[![npm version](https://badge.fury.io/js/%40peculiar%2Fasn1-cert-transparency.svg)](https://badge.fury.io/js/%40peculiar%2Fasn1-cert-transparency)

[![NPM](https://nodei.co/npm/@peculiar/asn1-cert-transparency.png)](https://nodei.co/npm/@peculiar/asn1-cert-transparency/)

ASN.1 schema for Certificate Transparency structures and signed certificate timestamp data defined in RFC 6962.

Use the exported classes with `@peculiar/asn1-schema` helpers such as `AsnConvert`, `AsnParser`, and `AsnSerializer` to parse or serialize DER-encoded data defined by the referenced specification.

## Installation

```bash
npm install @peculiar/asn1-cert-transparency
```

## Specifications

- [RFC 6962: Certificate Transparency](https://datatracker.ietf.org/doc/html/rfc6962)
